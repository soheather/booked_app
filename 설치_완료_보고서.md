## 🚀 빠른 시작 가이드

### 1단계: Homebrew, Git, Watchman 설치

터미널에서 실행:

```bash
cd ~/booked_app
./setup_commands.sh
```

이 스크립트는 다음 작업을 수행합니다:
- Homebrew 권한 수정 (sudo 비밀번호 입력 필요)
- Homebrew 업데이트
- Git 업데이트
- Watchman 설치

### 2단계: Xcode 설치

**App Store를 통한 설치 (권장):**

```bash
open -a "App Store"
```

- "Xcode" 검색 및 설치
- 약 15GB, 설치에 시간 소요
- 설치 후 실행:

```bash
sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer
sudo xcodebuild -license accept
```

**또는 Apple Developer에서 다운로드:**
- https://developer.apple.com/download/all/
- Xcode 16.2 검색 및 다운로드

### 3단계: Android Studio 설치

**Homebrew를 통한 설치 (권장):**

```bash
brew install --cask android-studio
```

**또는 직접 다운로드:**
- https://developer.android.com/studio

**설치 후 환경 변수 설정:**

`~/.zshrc` 파일에 추가:

```bash
export ANDROID_HOME=$HOME/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/emulator
export PATH=$PATH:$ANDROID_HOME/platform-tools
export PATH=$PATH:$ANDROID_HOME/tools
export PATH=$PATH:$ANDROID_HOME/tools/bin
```

적용:

```bash
source ~/.zshrc
```

### 4단계: 최종 검증

모든 설치가 완료되면:

```bash
cd ~/booked_app
./verify_environment.sh
```

## 💡 중요 참고사항

### npm 권한 문제 해결됨

이전에 발생했던 npm 권한 오류는 nvm을 통해 Node.js를 재설치하여 완전히 해결되었습니다.

### nvm 환경 로드

새 터미널을 열 때마다 nvm이 자동으로 로드됩니다. `~/.zshrc`에 다음이 추가되어 있습니다:

```bash
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
[ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"
```

### Expo CLI 버전 참고

Expo CLI는 54.0.20 버전이 설치되었습니다. 이는 54.0.30과 매우 가까운 최신 안정 버전입니다.
