## 🔧 Android Studio 최종 설정 방법

Android Studio는 설치되어 있지만, SDK 경로 설정이 필요합니다.

### 1단계: Android Studio에서 SDK 설치

1. Android Studio 실행
2. More Actions → SDK Manager
3. Android SDK가 설치되어 있는지 확인
4. 없다면 설치 (보통 `~/Library/Android/sdk`)

### 2단계: 환경 변수 설정

터미널에서 실행:

```bash
# .zshrc에 환경 변수 추가
cat >> ~/.zshrc << 'EOF'

# Android Studio
export ANDROID_HOME=$HOME/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/emulator
export PATH=$PATH:$ANDROID_HOME/platform-tools
export PATH=$PATH:$ANDROID_HOME/tools
export PATH=$PATH:$ANDROID_HOME/tools/bin
EOF

# 적용
source ~/.zshrc

# 확인
echo $ANDROID_HOME
```

### 3단계: 검증

```bash
cd ~/booked_app
./verify_environment.sh
```

## 🚀 React Native 개발 시작하기

이제 바로 개발을 시작할 수 있습니다!

### 새 프로젝트 생성

```bash
# Expo 프로젝트 생성
npx create-expo-app my-app

# 프로젝트로 이동
cd my-app

# 개발 서버 시작
npx expo start
```

### 실행 옵션

개발 서버가 시작되면:

- **`i`** - iOS 시뮬레이터 실행 ✅
- **`a`** - Android 에뮬레이터 실행 (SDK 설정 후)
- **`w`** - 웹 브라우저에서 실행 ✅
- **QR 코드 스캔** - Expo Go 앱으로 실행 ✅
