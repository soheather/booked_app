## 🔧 사용자가 직접 완료해야 하는 항목

다음 항목들은 sudo 권한 또는 수동 설치가 필요하여 사용자가 직접 실행해야 합니다:

### 1. Homebrew, Git, Watchman 설치/업데이트

터미널에서 다음 스크립트를 실행하세요:

```bash
cd ~/booked_app
./setup_commands.sh
```

또는 수동으로 실행:

```bash
# Homebrew 권한 수정 (sudo 비밀번호 필요)
sudo chown -R $(whoami) /opt/homebrew/Cellar

# Homebrew 업데이트
brew update

# Git 업데이트
brew upgrade git

# Watchman 설치
brew install watchman

# 버전 확인
brew --version
git --version
watchman --version
```

### 2. Xcode 16.2 설치

**방법 1: App Store (권장)**

```bash
# App Store 열기
open -a "App Store"
```

- App Store에서 "Xcode" 검색
- Xcode 다운로드 및 설치 (약 15GB, 시간 소요)
- 설치 완료 후 Xcode를 한 번 실행하여 추가 구성 요소 설치

**방법 2: Apple Developer 사이트**

1. https://developer.apple.com/download/all/ 접속
2. Apple ID로 로그인
3. "Xcode 16.2" 검색 및 다운로드
4. .xip 파일 압축 해제 후 Applications 폴더로 이동

**설치 후 필수 작업:**

```bash
# Xcode Command Line Tools 경로 설정
sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer

# 라이선스 동의
sudo xcodebuild -license accept

# 버전 확인
xcodebuild -version
```

### 3. Android Studio 2025.2.2.7 설치

**방법 1: Homebrew Cask (권장)**

```bash
brew install --cask android-studio
```

**방법 2: 직접 다운로드**

1. https://developer.android.com/studio 접속
2. "Download Android Studio" 클릭
3. .dmg 파일 다운로드 및 설치

**설치 후 필수 설정:**

1. Android Studio 실행
2. "Standard" 설치 선택
3. SDK 설치 경로 확인 (보통 `~/Library/Android/sdk`)
4. Android SDK Platform 및 Tools 설치

**환경 변수 설정:**

`~/.zshrc` 파일에 다음 내용 추가:

```bash
export ANDROID_HOME=$HOME/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/emulator
export PATH=$PATH:$ANDROID_HOME/platform-tools
export PATH=$PATH:$ANDROID_HOME/tools
export PATH=$PATH:$ANDROID_HOME/tools/bin
```

설정 적용:

```bash
source ~/.zshrc

# ADB 설치 확인
adb --version
```

## 📝 추가 참고사항

### nvm 환경 로드

새 터미널을 열 때마다 nvm을 사용하려면 `~/.zshrc` 파일에 다음이 포함되어 있어야 합니다 (설치 시 자동으로 추가됨):

```bash
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
[ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"
```

### 주의사항

1. **npm 권한 문제 해결됨**: nvm을 통해 Node.js를 설치하여 이전의 npm 권한 오류가 해결되었습니다.

2. **Xcode 및 Android Studio**: 두 도구 모두 크기가 크므로 (Xcode ~15GB, Android Studio ~5GB) 설치에 시간이 소요됩니다.

3. **버전**: Expo CLI는 54.0.20 버전이 설치되었습니다 (54.0.30과 유사한 최신 버전).

4. **터미널 재시작**: 환경 변수 변경 후 터미널을 재시작하거나 `source ~/.zshrc`를 실행하세요.
